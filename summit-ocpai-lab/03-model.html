Desplegar el modelo

Importar modelo

  • Para empezar a trabajar con el modelo de detección de imágenes, es necesario importar el repositorio de Git en el que está el código desarrollado. Para ello, hazx click en el icono de Git en el menú de navegación de la izquierda.

  • Introduce la URL del repositorio de Git:

    https://github.com/rhcs-workshops/summit-connect-madrid-openshift-ai-lab.git

  • Una vez el repositorio esté importado en el servidor de JupyterLab, accede a la carpeta lab y abre el modelo llamado accident-recog.ipynb.

Ejecutar modelo

  • El modelo está desarrollado en un Notebook de Jupyter. El modelo que se va a utilizar en este laboratorio se basa en el modelo YOLOv8, que se puede encontrar online en yolo8.

  • Para ejecutar el modelo, si nunca se han ejecutado las celdas del Notebook, haz click en Restart kernel and Run all Cells para ejecutar el modelo, y obtener el resultado.

  • Click en Restart, y espera hasta que el modelo termine de ejecutar.

  • En cuanto el modelo termina su ejecución, se puede observar el resultado. El modelo ha evaluado los daños del accidente, mostrando el porcentaje de precisión del modelo.
