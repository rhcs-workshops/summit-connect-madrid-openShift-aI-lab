Crear un custom workbench

En OpenShift AI, un workbench es un entorno de trabajo contenerizado que se ejecuta como un pod. Los workbench incluyen una colección de liberías comunmente utilizadas en entornos de AI/ML, así como un servidor de JulyperLab.

En este laboratorio se va a crear un workbench personalizado que incluya las liberías y paquetes necesarios para implementar un modelo de detección de imágenes. Para la creación de un workbench personalizado, se deberá crear e importar una imagen que contenga el entorno JupyterLab, así como cualquier software adicional necesario para desplegar el modelo.

Importar imagen personalizada

  • Para importar una imagen custom, en el panel de navegación de la izquierda, accede a Settings - Notebook images y luego Import new image

  • Completa el formulario con los siguientes datos:

    • Image location: quay.io/acidonpe/ocp-ai-insurance-model:1.0

    • Name: Custom insurance model lab

  • El resto de campos pueden dejarse en blanco.A continuación, haz click en Create.

Crear un proyecto de Data Science

En Red Hat OpenShift AI, un proyecto de Data Science es la manera de organizar los recursos cuando se trabaja en una aplicación de AI/ML. De manera similar a cómo se utilizan proyectos en OpenShift para otras cargas de trabajo, puedes usar proyectos de Data Science para organizar los diferentes componentes que necesitas para tus aplicaciones de IA.

Una vez la imagen personalizada esté subida, se creará un proyecto de Data Science con el que trabajaremos y desplegaremos el modelo de AI/ML.

NOTA: Si tu usuario es user1, el proyecto que crearás se llamará user1.

  • Haz click en Create.

Crear workbench

A continuación, se creará un entorno de trabajo dentro del proyecto creado anteriormente, y donde usaremos esta imagen.

  • Dentro del proyecto userX, selecciona Create a workbench.

  • Completa los campos con la siguiente información:

    • Name: Lab Workbench

    • Image selection: Custom AI image

    • Deployment size: Small

  • Para acceder al workbench, hay que autenticarse con las credenciales de OpenShift AI.
